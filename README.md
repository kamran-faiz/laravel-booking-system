# Booking System Project

## Overview
This is a booking system built with **Laravel** that manages customers, providers, and an admin panel. It includes role-based dashboards and basic booking functionality.

### Features
- **Customer**
  - Register, login, and logout
  - Create bookings with date, time, and provider selection
  - View all bookings and their status

- **Provider**
  - View bookings assigned to them
  - Accept or reject bookings
  - See customer details for each booking

- **Admin**
  - View all users
  - Update user roles (Customer / Provider / Admin)
  - Delete users

### Technologies
- Laravel
- PHP
- Blade Templates
- Bootstrap 5 (for styling)
- MySQL (database)
